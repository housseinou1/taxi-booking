# Taxi Booking App — Real-Time Ride Booking System

## Overview

This Taxi Booking App is a real-time web application built using Angular and Django.  
It allows users to log in, request rides, track drivers live, and receive notifications.

This project demonstrates frontend and backend integration, REST API usage, and responsive UI development.

---

## Features

- User Login and Authentication
- Ride Request System
- Real-Time Driver Tracking
- Live Notifications
- Responsive Web Interface
- Angular Frontend with Django Backend
